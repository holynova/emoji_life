# Emoji Life Timeline

一个单文件网页工具，用 emoji 生成你的人生时间线。  
A single-file web app to generate your life timeline with emojis.

## Quick Start / 快速开始

直接在浏览器打开 [index.html](/Users/sym/code/emoji_life/index.html)。  
Open [index.html](/Users/sym/code/emoji_life/index.html) directly in your browser.

## Features / 功能

- 年份输入、固定事件、自定义 emoji 事件  
  Year inputs, fixed events, and custom emoji events
- 连续年份折叠 / 展开  
  Collapse or expand continuous year ranges
- 一键复制纯文本  
  One-click copy as plain text
- 导出结果区域为图片  
  Export the result area as an image
